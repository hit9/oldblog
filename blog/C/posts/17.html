gdb使用入门

允许转载, 但转载请注明出处

Date:2012-09-25

gdb是C语言开发之调试必备利器!

比如我们调试如下程序: 

#include <stdio.h>
int main(int argc, const char *argv[])
{
    char a[] = "hello world";
    return 0;
}

首先我们编译时加上参数-g :

gcc test.c -o test -g

然后用gdb调试

gdb test
