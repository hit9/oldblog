format VS %


分享到：

Date:2012-09-01

格式化输出一个字符串可以

name='hello'
print "%s"%name

还可以:

name='hello'
print '{}'.format(name)

两者有一点区别,当name=(1,2,3)的时候,第一种必须这样打印才可以: 

name=(1,2,3)
print "%s"%(name,)

而第二种没问题
